Registration Status Definitions
  • Applied: User has applied to a workshop. Added automatically when they click Apply.
  • Accepted: User’s application is approved. Updated manually by an admin.
  • Attended: User attended the workshop. Updated manually by an admin.
  • Denied: User’s application or registration was rejected. Updated manually by an admin.
  • Cancelled: User or admin canceled the registration or application.
  • Registered: User registered for a workshop. Added automatically when they click Register.
  • Waitlisted: User registered but the workshop is full. Added automatically when they click Register.
  • Pending: Placeholder status. Admin updates as needed.
  • No-show: User did not attend the workshop. Updated manually by an admin.
